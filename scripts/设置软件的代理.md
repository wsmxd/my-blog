```bash
# 设置git的代理
git config --global http.proxy http://代理IP:端口
git config --global https.proxy http://代理IP:端口

# 设置wget的代理
nano ~/.wgetrc
# 设置 HTTP 代理
http_proxy = http://代理IP:端口
# 例如：http_proxy = http://127.0.0.1:7890
# 设置 HTTPS 代理
https_proxy = http://代理IP:端口
# 例如：https_proxy = http://127.0.0.1:7890
```
不知道什么情况我在代理软件中打开了全局模式，但是有一些软件还是很慢，我只好手动在设置一下指向我自己本地的代理http://127.0.0.1:7890。
